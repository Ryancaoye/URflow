### 完成的改动
1. **曲线/控制杆超出坐标轴框** — 根因: Mathf.Lerp/InverseLerp 会 clamp t 到 [0,1]，改为 LerpUnclamped + 手动除法

3. **自由拖拽不被UI阻挡** — GUIUtility.hotControl 锁定拖拽，MouseDown 直接用距离判断不限矩形
4. **Shift+拖拽水平吸附** — P1 锁 Y=0, P2 锁 Y=1
5. **Y 拖拽范围扩大** — -2 ~ 3

### 关键经验 (重要!)
- **Mathf.Lerp 会 clamp!** 需要超出范围时必须用 LerpUnclamped
- **Mathf.InverseLerp 也会 clamp!** 手动 (pos-min)/(max-min) 替代
- **GUIUtility.hotControl** 是 Unity Editor 中防止拖拽被其他 GUI 元素拦截的标准做法
- GL 绘制确实不受 GUI.BeginClip 裁剪（之前验证过，这次利用了这个特性）
- overflow 只控制布局间距，GL 绘制范围不受限
